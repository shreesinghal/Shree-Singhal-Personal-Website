# Project card images

Drop one file per project here. The ProjectCard renders these at a 4:3 aspect
ratio with `object-cover`, so any reasonably-sized landscape photo will look
fine. ~1200×900 px JPEG is a good default.

Expected filenames (match the `image` field in `content/projects.ts`):

- `renu.jpg` — ReNU wind turbine
- `backscatter.jpg` — Ambient backscatter research
- `drone-laser.jpg` — Drone tracking laser-charging capstone
- `mitosis.jpg` — Project Mitosis delta 3D printer
- `market-making.jpg` — DQN vs Q-learning market making
- `riscv.jpg` — Embedded design (FUSE / RISC-V datapath)

If any file is missing the card falls back to a captioned placeholder, so it
is safe to add images one at a time.
